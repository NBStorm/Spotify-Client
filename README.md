🛠️ Cài đặt
Yêu cầu
Node.js phiên bản 14 trở lên

npm hoặc yarn

Hướng dẫn
Sao chép repository:

bash
Sao chép
Chỉnh sửa
git clone https://github.com/NBStorm/Spotify-Client.git
cd Spotify-Client
Cài đặt các phụ thuộc:

Sử dụng npm:

bash
Sao chép
Chỉnh sửa
npm install
Hoặc sử dụng yarn:

bash
Sao chép
Chỉnh sửa
yarn install
Khởi chạy ứng dụng:

Sử dụng npm:

bash
Sao chép
Chỉnh sửa
npm start
Hoặc sử dụng yarn:

bash
Sao chép
Chỉnh sửa
yarn start
Ứng dụng sẽ chạy tại http://localhost:5173.
