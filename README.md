
# 🛠️ Cài đặt

## Yêu cầu
- **Node.js** phiên bản 14 trở lên.
- **npm** hoặc **yarn**.

## Hướng dẫn

### Sao chép repository:
```bash
git clone https://github.com/NBStorm/Spotify-Client.git
cd Spotify-Client
```

### Cài đặt các thư viện phụ thuộc:
Sử dụng **npm**:
```bash
npm install
```

Hoặc sử dụng **yarn**:
```bash
yarn install
```

### Khởi chạy ứng dụng:
Sử dụng **npm**:
```bash
npm start
```

Hoặc sử dụng **yarn**:
```bash
yarn start
```

Ứng dụng sẽ được khởi động tại địa chỉ mặc định:  
👉 http://localhost:5173
